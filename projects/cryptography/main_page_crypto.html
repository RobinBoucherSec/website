Chiffrement César

Basé sur une substitution simple : chaque lettre est remplacée par une autre selon un décalage fixe (monoalphabétique).
Par exemple, avec une clé de 3 : A → D, B → E, etc.

Vigenère

Basé sur une substitution variable : chaque lettre est décalée différemment selon une clé (polyalphabétique).
C’est une amélioration du chiffre de César, utilisant une clé textuelle pour décaler les lettres de manière variable.
Par exemple : "A" dans la clé représente le décalage de 0 (ou 65 selon l’encodage ASCII), "B" représente un décalage de 1 (ou 66), etc.
Plus robuste que le César, car il résiste mieux à l'analyse des fréquences des lettres.
