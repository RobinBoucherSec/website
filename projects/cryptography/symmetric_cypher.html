DES Encryption

Description DES: Data Encryption Standard est un algorithme historique qui découpe le message en blocs et les mélange avec la clé.

Implementation of Data Encryption Standard (DES) in ECB mode using PyCryptodome.

# DES Implementation
from Crypto.Cipher import DES
from Crypto.Util.Padding import pad, unpad

# 64-bit key (8 bytes)
cle = b'12345678'
message = b'HELLO123'

# ECB Mode
cipher = DES.new(cle, DES.MODE_ECB)
message_chiffre = cipher.encrypt(message)
print("Encrypted (hex):", message_chiffre.hex())

# Decryption
cipher2 = DES.new(cle, DES.MODE_ECB)
message_dechiffre = cipher2.decrypt(message_chiffre)
print("Decrypted:", message_dechiffre.decode())

Key Features:

  • Standard DES implementation
  • Uses PyCryptodome library
  • ECB mode demonstration

# Custom Block Cipher (excerpt)
def encrypt_block(block, subkeys):
    state = block
    for subkey in subkeys:
        state = permute_bits(state, permutation_table)  # Diffusion
        state = xor_bits(state, subkey)                 # Confusion
    return state

Manual Encryption Comparison

File: chapitre_6_et_7.py

Demonstrates weak vs. strong encryption modes with custom and DES implementations.

# Weak vs. Strong Encryption
def manual_encrypt(plaintext, key_bytes, perm_func, block_size=8):
    plain_bytes = plaintext.encode()
    padding_len = block_size - (len(plain_bytes) % block_size)
    plain_bytes += bytes([padding_len]) * padding_len
    
    blocks = [plain_bytes[i:i+block_size] for i in range(0, len(plain_bytes), block_size)]
    
    encrypted_blocks = []
    for block in blocks:
        transformed = xor_block(block, key_bytes)
        permuted = perm_func(transformed)
        encrypted_blocks.append(permuted)
    
    return b''.join(encrypted_blocks)
